# Pre-Handoff Checklist

- Active source PDF explicitly uploaded or named.
- qpdf check passed.
- veraPDF PDF/UA passed.
- Pinned WCAG profile passed when profile repository available.
- Metadata Info + XMP parity passed after final save.
- Catalog language present.
- Native text preservation passed.
- Links/widgets/annotations/images preserved.
- Alt text/artifacting audit passed.
- Table semantics audit passed where applicable.
- Contrast audit passed or exceptions documented.
- Visual QA/file-size checks passed.
- External axesCheck/PAC reported as PASS only if actually run.
- Package includes logs, reports, status JSON, rules copy, manifest, and checksums.
