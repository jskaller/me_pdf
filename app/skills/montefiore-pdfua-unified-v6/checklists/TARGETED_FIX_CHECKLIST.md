# Targeted Fix Checklist

Use targeted fixes only when the user asks for a specific issue or small issue set.

- Scope the active PDF and affected pages/spans/objects.
- Fix only the targeted issue unless a hard gate is broken by the fix.
- Preserve live text, tags, reading order, links, annotations, layout, and typography.
- Rerun relevant local gates after the fix.
- Include targeted before/after QA artifacts and a concise report.
