# Controlled Scope and Reference Policy

Only files in the active V6 quickstart paths are controlling.

Examples, visual crops, historical reference outputs, and any prior package artifacts are read-only reference material. They may be inspected for calibration or troubleshooting, but must not be processed, remediated, validated, or included in the current PDF status unless the user explicitly names one as an active source PDF.

Active source PDFs are only PDFs explicitly uploaded or explicitly named by the user for the current job.

The active V6 paths are:

- `01_rules/montefiore_pdfua_quickstart_v6_01_rules/`
- `02_scripts/montefiore_pdfua_quickstart_v6_02_scripts/`
- `03_skills/montefiore_pdfua_quickstart_v6_03_skills/skills/montefiore-pdfua-unified-v6/`
- `05_assets/montefiore_pdfua_quickstart_v6_05_assets/validation_profiles/`

The examples package is reference-only by default.
