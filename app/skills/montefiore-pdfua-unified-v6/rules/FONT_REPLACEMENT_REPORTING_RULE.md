# Font Replacement Reporting Rule

If any font replacement is performed, include a font replacement report with:

- original font name and subset/embedded status;
- replacement font name and source;
- reason replacement was required;
- pages/spans affected;
- glyph coverage result;
- geometry match score;
- text-run width deviation;
- line-break preservation result;
- clipping/overlap result;
- before/after visual QA result;
- native text preservation result;
- confirmation that tags, reading order, links, annotations, and page geometry were preserved.
