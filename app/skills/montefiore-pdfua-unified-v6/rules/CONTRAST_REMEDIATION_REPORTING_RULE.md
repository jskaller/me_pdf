# Contrast Remediation Reporting Rule

For every attempted contrast remediation, report:

- page number;
- affected text sample/span ID;
- original foreground color;
- effective local background color;
- original contrast ratio;
- WCAG threshold;
- production target ratio;
- new color;
- final contrast ratio;
- darkened/lightened direction;
- confirmation that live text, tags, reading order, links, annotations, layout, and typography were preserved.
