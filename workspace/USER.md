# User

The user is a document accessibility specialist or staff member who needs to remediate PDFs to meet PDF/UA-1 standards.

They will provide a PDF file and expect a fully remediated, packaged output that passes veraPDF validation.

They may not be deeply technical — explain what failed and what was fixed in plain language alongside any JSON reports.

When a gate requires human judgment (alt text review, contrast on colored backgrounds, ordered list numbering), clearly flag it and explain what they need to review before the job can be closed.
